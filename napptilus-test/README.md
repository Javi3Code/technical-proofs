## Napptilus Technical Proof - Spring Boot Project

## Prerequisites

To run this project, make sure you have the following software installed on your system:

```
- Java Development Kit (JDK) 17 or later
- Apache Maven
- Spring-boot 3.1
- Docker
- IDE (IntelliJ IDEA recommended)
```
